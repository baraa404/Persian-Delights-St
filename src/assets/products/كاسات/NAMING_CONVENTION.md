# Image Naming Convention

## Folder Structure
Each zip file creates a folder named after its code (e.g., `150.zip` → `150/`)

## Naming Format
```
[number]_[code].webp
```

## Examples (for code 150)
```
1_150.webp
2_150.webp
3_150.webp
4_150.webp
5_150.webp
```

## Process
1. Unzip file to folder with same name
2. Rename all images sequentially: `1_code.webp`, `2_code.webp`, etc.
3. Run `imgc [folder]` to compress all images to WebP
